# 🎬 影片生成功能 - 超簡單設置指南

## 📝 您需要做的事情（總共 3 步）

### ✅ 步驟 1：啟動 Python 後端服務（5 分鐘）

1. **打開終端機（Terminal）**

2. **進入 video-generator 資料夾**
```bash
cd video-generator
```

3. **執行啟動腳本**
```bash
./start.sh
```

4. **如果出現錯誤，手動執行這些命令：**
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```

5. **看到類似這樣的訊息就成功了：**
```
INFO:     Uvicorn running on http://0.0.0.0:8000
```

**⚠️ 重要：讓這個終端機視窗保持開啟！不要關閉它！**

---

### ✅ 步驟 2：配置 API Keys（10 分鐘）

1. **打開另一個終端機視窗**（第一個視窗不要關閉）

2. **編輯環境變數文件**
```bash
cd video-generator
nano .env
```
（如果沒有 .env 文件，先執行：`cp env.example .env`）

3. **填入您的 API Keys**（至少需要前 4 個）

您需要去這些網站註冊並獲取 API Key：

**必須要有的：**
- **Gemini**：https://makersuite.google.com/app/apikey
  - 複製 API Key，貼到 `.env` 文件的 `GEMINI_KEY=` 後面

- **ElevenLabs**：https://elevenlabs.io/
  - 註冊帳號，獲取 API Key，貼到 `ELEVENLABS_KEY=` 後面

- **HeyGen**：https://www.heygen.com/
  - 註冊帳號，獲取 API Key，貼到 `HEYGEN_KEY=` 後面

- **Creatomate**：https://creatomate.com/
  - 註冊帳號，獲取 API Key，貼到 `CREATOMATE_KEY=` 後面

**可選的：**
- **Kling AI**：https://www.klingai.com/
  - 如果有的話，貼到 `KLING_KEY=` 後面

4. **保存文件**
   - 在 nano 編輯器中：按 `Ctrl + X`，然後按 `Y`，然後按 `Enter`

5. **重新啟動 Python 服務**
   - 回到第一個終端機視窗
   - 按 `Ctrl + C` 停止服務
   - 再次執行：`uvicorn main:app --reload --host 0.0.0.0 --port 8000`

---

### ✅ 步驟 3：配置 Next.js 連接（2 分鐘）

1. **在專案根目錄創建或編輯 `.env.local` 文件**
```bash
cd ..  # 回到專案根目錄
nano .env.local
```

2. **添加這一行：**
```
PYTHON_API_URL=http://localhost:8000
```

3. **保存文件**（`Ctrl + X`，`Y`，`Enter`）

4. **重新啟動 Next.js 服務**
   - 如果 Next.js 正在運行，停止它（`Ctrl + C`）
   - 重新啟動：`npm run dev`

---

## 🎉 完成！測試功能

1. **打開瀏覽器**，訪問：http://localhost:3000/shorts

2. **點擊右上角的「Video Generator」按鈕**

3. **填寫表單：**
   - 職位描述（必填）
   - 企業 Logo URL（必填）- 例如：`https://example.com/logo.png`
   - 主管照片 URL（必填）- 例如：`https://example.com/manager.jpg`
   - 辦公室影片 URL（可選）

4. **點擊「開始生成影片」**

## ❓ 常見問題

### Q: 出現 "fetch failed" 錯誤？
**A:** 確認 Python 服務正在運行（步驟 1），檢查終端機是否有錯誤訊息

### Q: 出現 API Key 錯誤？
**A:** 檢查 `video-generator/.env` 文件中的 API Keys 是否正確填入

### Q: 端口 8000 被占用？
**A:** 使用其他端口，例如：
```bash
uvicorn main:app --reload --host 0.0.0.0 --port 8001
```
然後更新 `.env.local` 為：`PYTHON_API_URL=http://localhost:8001`

### Q: 不知道怎麼編輯文件？
**A:** 可以用任何文字編輯器打開 `.env` 和 `.env.local` 文件，例如：
- VS Code
- TextEdit（Mac）
- Notepad（Windows）
